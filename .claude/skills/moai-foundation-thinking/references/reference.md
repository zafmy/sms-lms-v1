## Framework Comparison

When to Use Which:

| Need | Philosopher | Thinking |
|------|------------|----------|
| Architecture decision | Primary | Supporting |
| Technology selection | Primary | Supporting |
| Idea generation | Not suited | Primary |
| Proposal evaluation | Supporting | Primary |
| Requirement analysis | Supporting | Primary |
| Cognitive bias check | Primary | Not suited |
| Creative exploration | Not suited | Primary |
| Trade-off analysis | Primary | Supporting |

Philosopher focuses on strategic decisions with explicit trade-offs.
Thinking focuses on creative exploration with rigorous evaluation.

Both complement each other for comprehensive analysis.
